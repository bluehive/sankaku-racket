# sankaku-racket

単位円の回転と sin / cos 波を「影（投影）」として結びつけるアニメーションを、**Racket 標準ライブラリのみ**で再現するプロジェクト。

目標: https://x.com/TheMathFlow/status/2070774628713451694 のような「円の影が波になる」三次元的な描写を、最初は 2D canvas + 手動投影で実現。

## 現在の状態（2026-07-04）

- `main.rkt` : **完成**（PLAN.md 承認済み仕様で実装）
  - 2D統合ビュー（円:左 / sin波:右縦 / cos波:下横）
  - 円内に直角三角形（O-Q-P）を描画し、点の移動で変形
  - 影の対応を明確にする射影線（水平=sin / 垂直=cos）
  - サイン波・コサイン波の両方に横軸（sin=0 基準線 / フロント水平線）を追加
  - 反時計回り回転で波を描画
  - 必須機能: 速度変更・一時停止・波の表示長調整（マウス操作は不要）
  - ラベル: 英語

PLAN.md に全仕様（モジュール・変数・関数名含む）が記載されています。

## 実行方法

```powershell
# Racket が PATH に通っている場合
racket main.rkt

# または DrRacket で main.rkt を開いて Run
```

Racket が入っていない場合は公式サイトから Windows 用をインストールしてください:
https://download.racket-lang.org/

## 改訂計画の概要

詳細は **[PLAN.md](PLAN.md)** を参照（仕様書レベルでモジュール・変数名・関数名を記載）。

### 確定レイアウト（ユーザー回答 + 追加仕様）
- 円 → 左側（内部に直角三角形 O-Q-P を描画、変形する）
- サイン波 → 円の右側（縦向き）＋ 横軸（sin=0 水平基準線）
- コサイン波 → 円の下側（横向き）＋ 横軸（現在時刻フロント水平線）
- 点の反時計回り移動で波を描画
- 2D統合ビュー + 明確な射影線 + 三角形で「影の対応」を視覚化

### 必須機能（v1）
- 速度変更 / 一時停止
- 波の表示長（周期数）調整
- キーボード操作のみ（マウス不要）
- ラベルは英語

新実装は PLAN.md の名前・ロジック・追加仕様（三角形・横軸）に完全に従っています。
